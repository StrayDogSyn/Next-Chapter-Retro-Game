# Audio Assets Naming Convention

Use CC0/open-source chiptune SFX and keep files short for responsive playback.

Recommended naming:

- `sfx-<action>.wav` (example: `sfx-jump.wav`)
- `music-<scene>.ogg` (example: `music-overworld.ogg`)

For the scaffold, `components/GameCanvas.tsx` attempts to load `/audio/jump-placeholder.wav`.
